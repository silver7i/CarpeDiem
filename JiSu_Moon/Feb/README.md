#### 월 (Month) 약어 (앞 3~4글자) 
1월: Jan. (January)     
2월: Feb. (February)        
3월: Mar. (March)       
4월: Apr. (April)       
5월: May (약어 없음)        
6월: Jun. (June)        
7월: Jul. (July)        
8월: Aug. (August)      
9월: Sep./Sept. (September)       
10월: Oct. (October)        
11월: Nov. (November)       
12월: Dec. (December)

#### 영문 날짜(일) 서수 약어
1일: 1st (First)
2일: 2nd (Second)
3일: 3rd (Third)
4일~20일: 4th, 5th, ..., 20th
21일: 21st
22일: 22nd
23일: 23rd
24일~31일: 24th, ..., 31st
