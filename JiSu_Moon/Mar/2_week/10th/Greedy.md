### 조합적 문제
#### 부분 집합
어떤 집합의 공집합과 자기자신을 포함한 모든 부분\
구하고자 하는 어떤 집합의 원소 개수가 n일 경우 부분집합의 수는 2n(제곱)개
-  집합에서 부분 집합을 찾아내는 구현 방법
1. 완전탐색
2. Binary Counting
```
arr = [1,2,3]

for i in range(1<<3):
    answer = []
    for index in range(3):
        if i &(1<<index) != 0:
            answer.append(arr[index])
    print(answer)
```

#### 조합
서로 다른 n개의 원소 중 r개를 순서 없이 골라낸 것

### 탐욕 알고리즘
#### Greedy
결정이 필요할 때, 현재 기준으로 가장 좋아 보이는 선택지로 결정하여 답을 도출하는 알고리즘

#### Knapsack 문제
무게 제한이 있는 배낭에 물건을 넣어서 가치의 합을 최대화하는 문제\
물건마다 무게(weight) 와 가치(value) 가 있음\
배낭은 최대 무게 W 까지만 담을 수 있음\
어떤 물건을 넣을지 선택해야 함\

1. 0/1 Knapsack\
물건을 하나 선택하면 쪼갤 수 없음\
선택 or 선택 안함\
👉 DP로 해결

2. Fractional Knapsack\
물건을 쪼개서 담을 수 있음\
👉 Greedy

#### 활동 선택 문제
겹치지 않는 최대 개수의 활동을 선택하는 문제\
끝나는 시간이 가장 빠른 활동부터 선택\
👉 Greedy
